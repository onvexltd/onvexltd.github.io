Linux Essentials

What is a Linux OS?

Most computers run on windows and MacOS. These two are the most popular operating systems in the markets today. An operating system basically allows users and computer hardware to communicate. A computer is basically useless without an operating system. Linux is distributed under an open source licence. This gives users the freedom to use and modify the software. Programmers have been able to design their own custom operating systems based on the Linux Kernel. There is a new range of user-friendly operating systems under the linux distribution. Examples are Ubuntu

Why Linux?

Most programmers love the linux operating system. Most non-tech people shun linux under the impression that it is complex to use. These days there are user friendly versions of linux that almost everyone can use. Why you should use linux. It has several advantages over windows

  • Secure
  • open-source, so you can alter your OS in any way you want.
  • Plenty of development applications built around Linux making it great for programmers

Must Know Linux Terms

If you are starting out in linux, here are smoe term you definitely need to know.

Distro - Distribution

When starting out in linux, you have to make a choice on what linux distribution to use. A distribution is commonly known as a distro. A distro is an implementation of the linux kernel into an operating system to suit different tastes and needs of a community. Linux is open source and therefore different people and organzations make different forms . Distributions can come in different GUI interfaces, drivers, application softwares , pacakage management systems .Varieties of distros are in the market today. Each looks different but are all based on the linux kernel. Some distributions are commercial while others are for community use. Heres a list of some popular desktop distros :

  • Linux
  • Ubuntu
  • Debian
  • Fedora
  • >KDE Neon
  • Parrot

GNU

GNU in full is GNUâ€™S Not Unix. GNU is basically a collection of open source software licenced under GPL(General Public License). GNU is an operating system and linux is a kernel. It was launched by Richard Stallman for the purpose of creating an operating system that is free software. More infrmation on the GNU project.

Gnome

GNU Network Object Model Environment is basically a GUI interface for linux users. It is useful in making the linux operating system more user friendly especially for non-technical folk. It gives users the freedom to choose their more likeable desktop appearances hence improving their general user experience.

Shell

Linux shell is a program that allows users to give commands to the OS to process and return an output.It lets the user interact with the kernel in a more simpler way. The most common shell in linux is bash.

Root

When we talk about root in linux we should think of the extent of permissions and privileges it has on the system. The root account or super user has full system privileges. The privileges include certain acesss and permissions, ability to modify and access system files.

Package manager

The package manager installs software in your machine. Instruct the package manager to download software through a set of commands and it will automatically install in your machine. It also keeps track of updates and installs them for you. Package managers simplifies the process of installing and updating softwares in linux With an idea of what a pacakage manager is, you probably have an idea of what a package is
